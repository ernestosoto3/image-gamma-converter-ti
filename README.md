# Python Gamma Image Converter

Simple macOS image gamma converter written in Python.

## Features
- No pip installs required
- Uses macOS built-in `sips`
- Supports custom gamma values
- Gamma < 1.0 makes image lighter
- Gamma = 1.0 makes no change
- Gamma > 1.0 makes image darker

## Run
```bash
python3 gamma_convert.py
